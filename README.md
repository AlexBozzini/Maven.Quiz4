* **Methods to Complete**
	* `Boolean hasVowels(String word)`
		* return `true` if `word` contains `a`, `e`, `i`, `o`, or `u`.

		* return `true` if `character` is `a`, `e`, `i`, `o`, or `u`

		* return the index of the first vowel occurring in `word`
	* `Boolean startsWithVowel(String word)`
